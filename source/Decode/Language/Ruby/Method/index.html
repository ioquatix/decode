def long_form

The long form of the method. e.g. def puts(*lines, separator: "\n").

Implementation

 
def long_form
	if arguments_node = self.arguments_node
		@node.location.keyword.join(
			arguments_node.location.expression
		).source
	else
		self.short_form
	end
end
